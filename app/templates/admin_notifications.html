Airspace summary

A periodic digest: aircraft count + a fun breakdown (helicopters / light / private jets / cargo / seaplanes), a “special news” line from triggers you mark Include in the airspace summary (falling back to “time since last <trigger>” after summary_news_lookback_hours), and an emergency line. Where it goes & how often is per target now: create a channel under Profile → channels, set its Mode = Summary, and pick its cadence (e.g. TRMNL every 7 min).
